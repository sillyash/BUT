## Setup

This project uses `uv`. [__Install `uv`__](https://docs.astral.sh/uv/getting-started/installation/)
(no root / admin acess required).

To install the dependencies in a virtual environment, use:

```bash
uv sync
```

You might also need to install `tkinter` which is a system package.
Please refer to official documentation. 

## Run

From the parent directory:

```bash
uv run TP3
```

Or the TP3 directory:

```bash
uv run .
```

## Help

From the parent directory:

```bash
uv run TP3 -h
```

Or the TP3 directory:

```bash
uv run . -h
```
