## Run

From the parent directory:

```bash
python3 TP2 assets/imagemystere.json
```

Or the TP2 directory:

```bash
python3 . assets/imagemystere.json
```

## Help

From the parent directory:

```bash
python3 TP2 -h
```

Or the TP2 directory:

```bash
python3 . -h
```
