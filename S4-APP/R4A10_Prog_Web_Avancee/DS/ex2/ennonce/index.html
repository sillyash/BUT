Calculette simplifiée

Ales-Bianchetti Jacques, IUT-Orsay

Ennoncé

La calculette

  1. Vous devez contruire une calculette très simpliée. Elle permet de :

    • Saisir des nombres entiers, chiffre par chiffre

    • Saisir des opérations parmis les quatres basiques : addition, soustraction, multiplication, division

    • De demander le calcul d'une expression ainsi construite

  2. Le comportement attendu est illustré par les captures suivantes

clic sur 1, puis 2

clic sur -, puis 2

clic sur =

clic sur Reset

Remarques

  • On supposera que l'expression saisie par l'utilisateur est correcte. Ainsi, par exemple, il ne demandera pas d'évaluer 3*/2-

    Vous pouvez donc vous disposer de controler l'expression saisie avant de l'évaluer

  • Vous pouvez utiliser la fonction eval() ; Celle-ci recoit une expression javascript, l'execute, et retourne la valeur correspondante

    Ainsi eval("2+2") retourne 4
