Remplissage du main

Ales-Bianchetti Jacques, IUT-Orsay

1) Ennoncé

  1. Vous disposez dans votre fichier index.js, contenant une variable contenu_du_main correspondant à un objet JSON. Cet objet décrit le contenu de la balise main que le script doit construire lors du chargement du document html

  2. Le document html continent la structure initiale requise : balises html, head, body, … Ces éléments ont tous les fils et attributs nécessaires pour la structure de base. Le document html contient une balise main, mais celle-ci n'a aucun enfant.

  3. L'objet contenu_du_main est contruit de la façon suivante :

    • Chaque balise standard est représentée par un dict dont les attributs sont :

      • tag : la string indiquant le tag de la balise

      • attributs : la liste des paires [ nom d'attribut, valeur ] portés par la balise

      • fils : la liste des enfants de cette balise

    • Chaque contenu textuel, comme par exemple le contenu d'un paragraphe p ou d'un titre h1, est représenté par un dict ayant un seul attribut :

      • texte : la string indiquant le texte voulu

  4. Par exemple, on peut avoir la valeur suivante : 

    let contenu_du_main = {
   tag       : 'main',
   attributs : [ ['class','joli'] ],
   fils      : [
      {
         tag       : 'h1',
         attributs : [],
         fils      : [
            { texte : 'titre un' }
         ]
      },      
      {
         tag       : 'section',
         attributs : [],
         fils      : [ 
            {
               tag       : 'article',
               attributs : [],
               fils      : [ 
                  {
                     tag       : 'h2',
                     attributs : [],
                     fils      : [ 
                        { texte : 'titre deux' }
                     ]
                  },
                  {
                     tag       : 'p',
                     attributs : [],
                     fils      : [
                        { texte : 'paragraphe un' }
                     ]
                  },
                  {
                     tag       : 'p',
                     attributs : [],
                     fils      : [
                        { texte : 'paragraphe deux' }
                     ]
                  },
               ]
            },
            {
               tag       : 'article',
               attributs : [ ['class','attention'] ],
               fils      : [ 
                  {
                     tag       : 'p',
                     attributs : [],
                     fils      : [
                        { texte : 'paragraphe trois' }
                     ]
                  },
               ]
            },
         ]
      },      
   ]
};
