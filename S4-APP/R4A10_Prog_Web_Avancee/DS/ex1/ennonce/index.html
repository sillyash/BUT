tableau éditable

Ales-Bianchetti Jacques, IUT-Orsay

1) Enoncé

  • Le but de cet exercice est de rendre les cases d'un tableau éditables . Pour cela :

    1. l'utilisateur doit cliquer sur la case dont il veut modifier le contenu

    2. cette case doit devenir une zone de saisie ; et l'utilisateur saisit alors la nouvelle valeur voulue

    3. la saisie se conclut par l'appui de la touche entrée ; la zone de saisie est alors remplacée par une case normale, contenant la nouvelle valeur

  • Un contrôle supplémentaire est demandé. Les cases sont toutes prévues pour contenir des nombres entiers entre 0 et 20. Dans le cas où la nouvelle valeur ne respecterait pas cela, la case doit prendre la classe css incorrecte

  • Si par la suite une nouvelle valeur correcte est saisie, tout doit rentrer dans l'ordre
